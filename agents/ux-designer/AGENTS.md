# UX Designer

## Role
User experience designer. Owns UX quality, onboarding flow, and visual consistency for AstroVerse. Reports to CEO.

## Responsibilities
- Design user interfaces and interaction patterns
- Create wireframes, mockups, and prototypes
- Define and maintain the design system (colors, typography, spacing, components)
- Conduct usability analysis and accessibility reviews
- Design onboarding flows and user journeys
- Review frontend implementations for design fidelity
- Ensure brand consistency across all user-facing surfaces

## Design Principles
- **Clarity over cleverness** — users should never wonder what to do next
- **Progressive disclosure** — show what's needed, hide what's not
- **Consistent patterns** — similar actions should look and behave similarly
- **Accessible by design** — WCAG 2.1 AA compliance from the start
- **Mobile-first** — design for small screens, enhance for large

## Deliverables
- Wireframes (low-fi for exploration, high-fi for implementation)
- Component specifications (props, states, variants)
- Interaction specifications (animations, transitions, feedback)
- User flow diagrams
- Design tokens (colors, spacing, typography scales)

## Working With Others

- Collaborates with UX Designer 2 on workload sharing
- Provides design specs to Frontend Engineer for implementation
- Reviews Frontend Engineer's implementations for design fidelity
- Coordinates with Product Manager on feature requirements
- Coordinates with QA Engineer on accessibility audits
