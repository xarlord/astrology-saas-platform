- Mini chart: `scaleIn` (0.95 → 1) with 0.2s delay
- Transit list items: `stagger` fade-in, 0.05s between items

### Chart Wheel Visual Spec

- **Container:** Glass card, `border-radius: 1rem`, centered
- **Background:** Subtle radial gradient (cosmic purple glow at center)
- **Wheel rings:** 3 concentric — outer (houses), middle (zodiac signs), inner (aspects)
- **Planets:** Color-coded dots with glyph labels, 12px size
- **Aspect lines:** Colored by aspect type, 1px stroke, semi-transparent
- **Hover planet:** Enlarge to 16px, show tooltip with sign/degree/house
- **Click planet:** Scroll to planet detail in side panel

### Animation Spec

- Chart wheel: `draw` animation on mount — planets appear sequentially (0.1s stagger)
- Aspect lines: fade in after planets (0.3s delay)
- Tab content: `fadeIn` on switch
- Planet list: `stagger` fade-in from top

## Shared Page Patterns

### Page Header (all pages)

- Height: auto (content-driven)
- Padding: `xl` horizontal, `lg` vertical
- Background: transparent (inherits page bg)
- Back button: `Button` ghost with arrow-left icon
- Actions: right-aligned button group

### Page Transition (all pages)

- Route change: current page `fadeOut` (150ms) → new page `fadeInUp` (300ms)
- Shared layout (sidebar): no transition (persistent)
- Tab switch: content `fadeIn` (150ms), no page transition

### Loading States (all pages)

- Initial load: `SkeletonScreen` matching page layout
- Chart load: `SkeletonChartWheel` with shimmer
- Data refresh: inline `LoadingSpinner` (sm) in section header
- Error: `ErrorRecovery` with retry button

### Empty States (all pages)

- Centered vertically in content area
- Illustration (cosmic-themed) at 120px
- Headline in Space Grotesk 600
- Description in Noto Sans 400
- CTA `Button` primary
