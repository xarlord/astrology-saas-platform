## 5. Effects & Treatments

### Glassmorphism (Signature Style)

The glass effect is AstroVerse's visual signature. Apply to cards, modals, and overlay panels.

```css
/* Standard glass card */
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.1);
border-radius: 1rem;
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);

/* Hover state */
background: rgba(255, 255, 255, 0.10);
border-color: rgba(255, 255, 255, 0.15);
```

**Rules:**
- Always apply over a non-flat background (gradient, image, or deep-dark surface)
- Border must be subtle — never opaque
- Content on glass must meet contrast requirements (WCAG AA)
- Avoid stacking glass-on-glass (max 1 layer)

### Glow Effects

```css
/* Standard glow — primary purple */
box-shadow: 0 0 20px rgba(107, 61, 225, 0.3);

/* Large glow — emphasis */
box-shadow: 0 0 40px rgba(107, 61, 225, 0.4);

/* Inner glow — subtle depth */
box-shadow: inset 0 0 20px rgba(107, 61, 225, 0.1);
```

**Usage:** Active nav items, focused inputs, premium badges, chart highlights.

## 7. Iconography

### Icon Library: Material Symbols Outlined

**Usage rules:**
- Default size: 24px
- Dense contexts: 20px
- Featured/hero: 32px or 40px
- Color inherits from parent text color
- Exception: planet and zodiac glyphs use their assigned colors
- Optical weight: use 400 weight for standard, 700 for emphasis

### Zodiac Iconography

Each zodiac sign should have a consistent glyph representation:
- Stroke-based line art (not filled)
- 2px stroke weight at 24px
- Rendered in element color by default, white on dark backgrounds
- Animated hover: subtle rotation (5deg) or scale (1.05)

### Planet Glyphs

- Single-weight stroke style matching zodiac glyphs
- Color-coded per planet color table
- Size: 20px inline, 32px standalone, 48px in chart wheels

## 8. Dark/Light Theme Behavior

### Dark Theme (Default)

- Background: deep navy to near-black gradient
- Text: white primary, gray-400 secondary
- Cards: glass effect with semi-transparent backgrounds
- Borders: subtle white transparency (0.1 alpha)
- Accents: vivid (no dimming needed on dark)

### Light Theme

- Background: near-white to light gray
- Text: gray-900 primary, gray-500 secondary
- Cards: white with subtle shadow (no glass effect)
- Borders: gray-200
- Accents: same hues, slightly desaturated if needed for contrast

**Transition between themes:** 300ms ease on all color properties.
