- **Modal**: No `nested modal` stacking guidance
- **Dropdown**: No `search/filter` variant
- **ZodiacBadge**: Only has default state — needs hover, selected, disabled
- **LoadingSpinner**: Missing `skeleton` integration pattern
- **ChartWheel**: Loading/error states not formalized
- **Toast**: Missing `action` variant (toast with undo button)

## 6. Inconsistencies Noted

1. **Dual ChartWheel** — exists in both `components/ChartWheel.tsx` and `components/astrology/ChartWheel.tsx`
2. **Dual ZodiacBadge** — same pattern, two locations
3. **Dual store dirs** — `store/` and `stores/` with overlapping files
4. **Legacy pages** — `LoginPage.tsx`, `RegisterPage.tsx`, `SynastryPage.tsx` still in codebase but not routed
5. **Mixed styling** — LandingPage uses inline styles + CSS vars; internal pages use Tailwind dark: classes; design system components use Framer Motion
6. **No shared focus ring** — each component implements its own focus styling rather than using a shared utility class
