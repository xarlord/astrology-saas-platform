# @astrology-saas/shared-utils

Shared utility functions used across the AstroVerse monorepo.

## Installation

```bash
npm install @astrology-saas/shared-utils
```

## Exports

| Export | Kind | Description |
|--------|------|-------------|
| `API_BASE_URL` | `const string` | Default API base URL (`'http://localhost:3001'`) |

## Usage

```typescript
import { API_BASE_URL } from '@astrology-saas/shared-utils';
```
